## 阶段一：构建与部署

### 1.1 Windows 客户端
- [ ] **创建/替换托盘图标文件**（当前使用占位符）
  - 设计或下载 `icon_waiting.ico`（蓝色）
  - 设计或下载 `icon_connected.ico`（绿色）
  - 设计或下载 `icon_error.ico`（红色）
  - 设计或下载 `app.ico`（应用图标）

- [ ] **Windows 编译测试**
  - 在 Windows PC 上 clone 仓库
  - 运行 `dotnet build` 验证编译
  - 运行 `dotnet run` 启动应用
  - 检查托盘图标是否正常显示
  - 检查 QR 码窗口是否正常显示

- [ ] **Windows Firewall 配置**
  - 添加端口 8765 入站规则
  - 验证局域网内可访问

### 1.2 iOS App
- [ ] **创建 Xcode 项目**
  - 在 Xcode 中创建新项目（保存到 `ios/PhoneTypist`）
  - 导入所有 Swift 源文件
  - 配置 Info.plist（权限描述）
  - 添加 Starscream 依赖（SPM）

- [ ] **iOS 编译测试**
  - 构建项目，检查编译错误
  - 修复任何依赖或导入问题

- [ ] **iOS 真机测试**
  - 部署到 iPhone 设备
  - 授权麦克风、语音识别、相机权限
  - 验证 UI 显示正常

## 下一步立即行动

1. **在 Windows PC 上**：
   ```powershell
   git clone git@github.com:hufei365/phonetypist.git
   cd phonetypist/windows/PhoneTypist
   dotnet build
   ```

2. **在 Mac 上**：
   - 打开 Xcode
   - 创建 iOS 项目
   - 导入 Swift 源文件
   - 添加 Starscream 依赖
